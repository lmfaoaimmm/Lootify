# Lootify
Lootify is a modern gaming services marketplace with a focus on fast delivery, secure Stripe payments, and simple Discord-based order handling.


## 🚀 Overview

Lootify allows users to purchase gaming services such as:

- 🎮 Fortnite 1v1 Coaching
- ⚔️ Ranked Carries
- ⚙️ PC Optimization (Coming Soon)

The platform is designed to be:
- fast
- simple
- easy to use
- focused on real transactions

## 💳 Payments

Payments are handled securely using Stripe payment links.

Users:
1. Select a service
2. Confirm details via Discord
3. Complete payment through Stripe
4. Receive service

## 💬 Order Flow

Lootify uses Discord as its backend system:

1. User clicks "Contact on Discord"
2. Order is confirmed
3. Payment is completed
4. Service is delivered

## 🛠️ Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Stripe (payments)
- Discord (order handling)
- GitHub Pages (hosting)
